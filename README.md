# Hak Ediş & Tazminat Analizörü

Bu proje, siber operasyon merkezi (SOC) arayüz temasına sahip, modern ve kapsamlı bir finansal hak ediş ve tazminat hesaplama aracıdır.

## Özellikler
- **Hesaplama Verileri Paneli:** Net maaş, zam oranı, fazla mesai saatleri ve katsayıları ile çalışılan süre (yıl/ay) girdileri.
- **Hak Ediş Detayları Paneli:** Tahmini brüt maaş, SGK işçi primi kesintisi (%14), işsizlik sigortası kesintisi (%1), kıdem tazminatı ve ihbar tazminatı çıktıları.
- **Dinamik Animasyon:** Sonuç ekranı "Hesapla" butonuna basıldığında terminal efektiyle açılır.
- **Gelişmiş CSS Mimari:** Ekran boyutları ne olursa olsun kaymayı önleyen ve yan yana düzeni koruyan kilitli esnek (`flex`) yapı.

## Kurulum ve Kullanım
1. ZIP dosyasını açın.
2. `index.html` dosyasını herhangi bir modern web tarayıcısında çift tıklayarak çalıştırın.
3. GitHub üzerinde barındırmak için dosyaları doğrudan repoya yükleyip GitHub Pages özelliğini aktif edebilirsiniz.
